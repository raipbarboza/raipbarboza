> Em transição para dados com propósito — coleto, estruturo e analiso métricas ESG de  
> companhias brasileiras a partir de fontes públicas.  
> Cada repositório aqui é um pedaço de um pipeline maior: da coleta bruta ao risco de carbono.

---

> Meu foco é construir infraestrutura de dados para sustentabilidade corporativa no Brasil —  
> conectando CVM, IBAMA, B3 ISE e relatórios GRI num único pipeline analítico orientado ao SBCE.

---

### `// projetos — um pipeline, três camadas`

**[python-esg-studies](https://github.com/raipbarboza/python-esg-studies)** `fundação`  
Aprendizado de Python aplicado a dados ESG reais — cálculo de emissões, scoring de empresas e métricas de sustentabilidade. A base conceitual de tudo.  
`Python` `Jupyter` `Pandas`

---

**[esg-web-scraping](https://github.com/raipbarboza/esg-web-scraping)** `em progresso`  
Coleta, armazenamento e análise de dados ESG públicos da Carteira ISE B3 — o principal índice de sustentabilidade empresarial do Brasil. Scraping → PostgreSQL → Pandas.  
`BeautifulSoup` `Requests` `PostgreSQL` `SQL` `Pandas` `B3 ISE`

---

**[CarbonTrack BR](https://github.com/raipbarboza/CarbonTrackBR)** `construindo`  
Pipeline de dados públicos para análise de risco de carbono de companhias abertas brasileiras. Integra CVM, IBAMA, B3 ISE e relatórios GRI/CDP — alinhado ao SBCE.  
`CVM` `IBAMA` `GRI` `CDP` `SBCE` `Python`
